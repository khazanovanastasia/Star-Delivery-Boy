
# STAR DELIVERY BOY — SHOWCASE

**Rogue-like survival RPG with unique pixel art style.**  
This public repository contains a curated selection of scripts, procedural systems, and media that illustrate the core mechanics of the game.

> *Full Unity project remains private.*

## About the Game

**Star Delivery Boy** puts the player in an anti-utopian city where they work as a "courier" of stars. The game explores survival, resource management, and character adaptation in a hostile environment.

- Pixel-art style, fully created by me
- Solo project (all 2D/3D assets, animations, and code done personally)
- Procedural generation of maps
- Character stats influence gameplay (humility, agility, imagination, intuition, charisma)
- Needs management (health, hunger, craving) and interactions with NPCs

## Core Gameplay Features

### Procedural Map & Daily Cycle
- Home base with storage, bed, and exit
- Street sections with shops, NPCs, and events
- Procedural map generation each day

### Character System
- Stats affected by clothing, inventory, and items consumed
- Daily goals, survival constraints
- RPG-style modifiers, balancing needs and risks

### Inventory & Items
- Items have temporary or permanent effects
- Stars as special items to distribute while avoiding detection
- Simple interaction system for pickups, usage, and storage

### Visuals & UI
- Pixel art assets
- UI reflecting health, hunger, craving, and stats
- Planned shaders for dynamic effects
